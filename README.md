# Kitchen Performance Audit System

A professional, chef-led audit and reporting tool designed to capture operational insights on-site and instantly generate high-quality consultancy reports.

Built for chefs, consultants, and hospitality operators who want to improve food margin, reduce waste, and drive kitchen performance.

---

## 🚀 What This Tool Does

This app allows you to:

* Capture audit data during a site visit
* Record key operational issues:

  * Waste
  * Over-portioning
  * Ordering & stock control
  * Kitchen layout
  * Leadership & systems
* Automatically calculate:

  * GP (Gross Profit)
  * Waste %
  * Priority score
* Instantly generate a **professional Post-Visit Report** including:

  * Detailed findings
  * Clear prioritised action plan
  * Immediate quick wins
  * Long-term improvement strategy

## 🛠 How to Use

### 1. Open the App

Simply open `index.html` in your browser or deploy via GitHub Pages.

### 2. Input Audit Data

Fill in:

* Site details
* Commercial data (sales, food cost, labour)
* Operational observations
* Waste, portioning, and ordering issues
* Kitchen layout notes

### 3. Generate Report

Click **"Generate Report"** to instantly create a structured consultancy report.

### 4. Export or Save

* Export report as HTML
* Print / Save as PDF
* Save audit as JSON to reload later

---

## 🌐 Deploying with GitHub Pages

1. Create a new repository (recommended name: `kitchen-performance-audit`)
2. Upload `index.html`
3. Go to **Settings → Pages**
4. Select branch: `main` and folder: `/root`
5. Your app will be live at:

```
https://your-username.github.io/kitchen-performance-audit/
```
